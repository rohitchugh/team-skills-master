## Deliverability Checklist

- [ ] SPF record set on sending domain
- [ ] DKIM signing enabled
- [ ] DMARC policy set (p=quarantine minimum)
- [ ] Sending domain warmed up (50/day → scale over 4 weeks)
- [ ] Unsubscribe link in every email
- [ ] Physical mailing address in footer
- [ ] List cleaned monthly (bounces + 90-day inactives)
- [ ] Test in Mail-Tester.com before major sends (aim for 9+/10)
- [ ] Segment lists by niche + engagement level
