# Weekly Review

Run as a team. Copy per week. Built on the ops-manager cadence; this version is tuned to inboxing.

```
Week of: __________   Lead this week: __________
```

## Monday — Review & Plan
```
□ Pull last week EPC + RPE by domain → table below
□ Flag any domain 3-day EPC < 0.35 → rotation review
□ Email stats: open%, CTR, complaint%, unsub% per major send
□ Deliverability: any placement/reputation slip in Postmaster?
□ Set top 3 priorities for the week

Domain        | 3-day EPC | RPE | Open% | Comp% | Status   | Action

______________|___________|_____|_______|_______|__________|________

Top 3 priorities:
1. ____________________
2. ____________________
3. ____________________
```

## Wednesday — Mid-week check
```
□ Any domain needing urgent rotation?
□ Campaigns pacing on plan? complaints under control?
□ Tech issues (forms, tracking, redirects, render)?
□ Anyone blocked / need a hand?
Notes: ____________________
```

## Friday — Wrap & Handoff
```
□ What moved this week (sends, page changes, rotations, wins)
□ What's blocked / pending
□ Trackers updated (performance + domain + list-health)
□ Next week scheduled
□ Scorecard note for anyone who leveled up something

Handoff summary (a person out next week could continue from this):
____________________
```
